This is an example project that will appear in your projects list because `published: true` is set.

## How to use

You can write your project documentation in Markdown, and it will be rendered with the project layout.

### Features

- Full Markdown support
- Syntax highlighting for code
- Embedded media support
- Custom styling

### Example code

```javascript
const greeting = "Hello, world!";
console.log(greeting);
```

## Interactive content

You can also embed HTML for interactive demos:

<div style="padding: 2rem; background: #f5f5f5; border-radius: 0.5rem; margin: 2rem 0;">
  <p style="margin: 0;">This is custom HTML content within your project.</p>
</div>

You can also use the `full-width` class for content that should span the full viewport width.
