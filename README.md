# dlimiter.net

Personal website of Dave Lim, built with Jekyll and deployed on GitHub Pages.

## Development

### Prerequisites

- Ruby 2.6.3 or higher
- Bundler

### Setup

```bash
# Install dependencies
bundle install

# Run local development server
bundle exec jekyll serve

# Build the site
bundle exec jekyll build
```

## Deployment

The site automatically deploys to GitHub Pages when changes are pushed to the `master` branch.

### Custom Domain

The site uses a custom domain (`dlimiter.net`) configured via the `CNAME` file. GitHub Pages automatically handles SSL/TLS certificates.
